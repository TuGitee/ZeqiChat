# ZeqiChat

择栖Chat，在线聊天系统，支持图片、语音、文件等内容上传，同时允许发表动态及评论。

## 安装依赖

```shell
$ npm i
$ npm install
```

## 前后端分离

### 前端
```shell
$ npm run serve
```

### 后端
```shell
$ npm run serve
$ npm run start
```
